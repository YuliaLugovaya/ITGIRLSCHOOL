Неделя 2


Вопросы


1. Что такое валидация?
Валидация – проверка кода на соответствие определённым правилам. Валидный код отображается предсказуемо и загружается быстрее невалидного. 
Валидность влияет на восприятие страниц поисковыми системами.

2. Какой спецсимвол используется для указания знака ©?
&copy;

3. Найдите пример еще тега HTML5, который не вошел в урок.
<object> - контейнер для встраивания мультимедиа.

4. Какой тег мы используем для создания меню?
<nav>

5. Чем плохо использовать div для всего?
Верстка с использованием только тега div не позволяет создавать семантически правильный код. Следует применять семантические теги (header, main, footer и другие) везде, 
где это возможно. А если возможности нет, использовать тег div. Это необходимо для поисковых систем и читабельности кода, для обеспечения доступности кода: если система знает, 
что это за блок, он приобретает смысл, то есть семантичность.

6. Какой атрибут выключает звук при воспроизведении видеофайла?
Атрибут muted.

7. Как сделать блок радиокнопок, чтобы выбирался только один пункт?
Использовать для тега input атрибут type со значением radio.

8. Каким тегом можно сделать поле ввода даты?
Тег input с атрибутом type и значением date: <input type="date">

9. Как определяется, какой именно пункт будет выбран по умолчанию в выпадающем списке?
С помощью атрибута selected.
<option value="1" selected="selected">Пункт 1</option>

10. Что такое хорошая разметка?
Хорошая разметка правильно отображается, а значит, работает.
Она простая и понятная: в ней минимально необходимое количество тегов и нет ничего лишнего, все имена классов читаются и понятно, для чего эти элементы нужны. 
Блоки разделены по смыслу.
Соблюдается определенный стиль написания кода, который принят в компании.
Разметка проходит проверку на валидность.
