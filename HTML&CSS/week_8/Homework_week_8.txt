Вопросы


1. Какие основные преимущества Bootstrap?
- высокая скорость создания качественной адаптивной вёрстки, это достигается благодаря использованию готовых компонентов, созданных профессионалами;
- кроссбраузерность и кроссплатформеннось (корректное отображение и работа сайта во всех поддерживаемых этим фреймворком браузерах и операционных системах);
- наличие большого количество готовых хорошо продуманных компонентов, протестированных сообществом веб-разработчиков на различных устройствах;
- возможность настройки под свой проект, достигается это посредством изменения SCSS переменных и использования Bootstrap миксинов (можно изменить количество колонок, цвета, 
отступы между колонками и многое другое);
 - низкий порог вхождения - для работы с фреймворком достаточно знать основы HTML, CSS, JavaScript и jQuery;
- наличие хорошо продуманного дизайна компонентов и согласованности (в Bootstrap все компоненты выполнены в едином стиле);
- наличие огромного сообщества, большого количества статей, рецептов и видеоматериалов.
Фреймворк Bootstrap – это проект с открытым исходным кодом, доступным на Github. Он имеет лицензию MIT. Это означает, что его можно использовать бесплатно как в личных, так и 
в коммерческих проектах.

2. Как сделать желтую кнопку с помощью Bootstrap?
Добавить в код строчку
<button type="button" class="btn btn-warning">Текст</button>
где класс btn-warning автоматически задает желтый цвет для кнопки.

5. Как убрать поля между колонками?
Пробелы между колонками в предустановленных классах можно удалить с помощью класса .no-gutters, который удаляет отрицательный margin из .row и горизонтальный padding из всех колонок.
.no-gutters {
 margin-right: 0;
 margin-left: 0;
 > .col,
 > [class*="col-"] {
   padding-right: 0;
   padding-left: 0;
 }
}
Если нужно создать дизайн «edge-to-edge», когда контент плотно прилегает к краям девайса, следует убрать родительский класс .container или .container-fluid.
Кроме того, отступы можно убирать и с помощью стилей css.

6. Как сделать навигацию на Bootstrap?
С помощью компонентов Навигация (класс .nav) или Навигационная панель (класс .navbar).
Так как базовый компонент класса .nav создан на флексбоксе, он обеспечивает хорошую базу для создания всех типов навигационных компонентов.
Навбары включают поддержку брендинга, навигации, классы цветовых схем, плагины свертывания, формы и многое другое. В практическом задании 2 я делала навигацию с помощью .navbar.

7. Что такое колонка auto-layout?
Это авторазметка столбцов. При таком подходе получаются безразмерные колонки, которые дают больше гибкости при разработке шаблонов. Колонки могут “расти” и “урезаться” по ширине. 
Используются классы колонок со специальными контрольными точками (например, .col-sm-6) для легкого расположения колонок без использования явно обозначенных номеров классов.
Авторазметка также означает, что можно задать ширину одного столбца и автоматически изменить размер столбцов вокруг него, использовать предопределенные классы сетки, миксины сетки 
или встроенные значения ширины.

8. Как сделать центрирование содержимого по горизонтали? А по вертикали?
Bootstrap основан на «флексах» и дарит нам все его возможности, которые доступны в простых готовых классах, в том числе вертикальное и горизонтальное выравнивание.
Центрирование по вертикали - align-items center:
<div class="row align-items-center">
         <div class="col">Текст</div>
</div>
Либо:
<div class="row">
         <div class="col align-self-center">Текст</div>
</div>
Центрирование по горизонтали - justify-content center:
<div class="row justify-content-center">
         <div class="col">Текст</div>
</div>

9. Как задать адаптивность для колонки шириной в 50%, чтобы при ширине экрана меньше 768px она становилась шириной на весь экран?
Прописать в коде брейкпойнт col-md-6. Это означает использование 6 колонок из 12, то есть 50% ширины на размерах экранов, которые больше или равны 768px. На экранах меньше, чем 768px, 
колонка станет шириной 100% и встанет вертикально:
<div class="container">
  <div class="row">
     <div class="col-md-6">Column </div>
  </div>
</div>

10. Что произойдет, если колонок станет больше 12?
Когда элементы колонок в одном ряду превышают количество двенадцати, происходит вертикальное сталкивание, или “обертывание”. Процесс известен как “враппинг колонок” (перенос колонок). 
Колонки в одной строке располагаются вдоль горизонтально, а затем встают вертикально вниз. Через каждые 12 элементов колонки в одной строке переходят на следующую строку.

12. Чем отличается container от container-fluid?
container устанавливает максимальную ширину max-width в каждом брейкпойнте. Это контейнер с фиксированной шириной для центровки контейнера по середине шаблона.
container-fluid имеет width: 100% на всех точках. Это контейнер с шириной во весь экран.
container масштабируется адаптивно по ширине экрана, так что в конце концов он может стать шириной на весь экран, как и container-fluid, но на маленьких устройствах.
