5. Ответить на вопросы:
a. Что такое атрибут тега? Какие есть атрибуты у тега img?
Атрибут тега – это специальное свойство внутри тега, которое обозначается особенным словом и добавляет определенные настройки тегу, то есть влияет на его параметры. Атрибуты пишутся внутри тега через пробел. Есть универсальные атрибуты, которые можно написать к любому тегу, например, атрибут style. А есть специфичные. Для тега <img> существуют такие специфичные атрибуты - src и alt:
<img src=”(ссылка на изображение)” alt “(название изображения)”>
b. Зачем нужен тег span?
Тег <span> – это блок, контейнер для текста, строчный элемент. Применяется к словам, фразам, находящимся в пределах абзаца. Позволяет выделить часть текста цветом, изменить шрифт, размер (придать определенный стиль с помощью атрибутов).
c. Как добавить иконку?
Чтобы добавить иконку, нужно в блоке head прописать тег <link> c атрибутами rel, href и type, например:
<link rel="shortcut icon" href="(ссылка на иконку)" type="image/png"/>
Иконку можно искать на сайтах наподобие flaticon.com.
d. Как сделать маркированный список?
Маркированный список создается с помощью тегов <ul> и <li> в следующем виде:
<ul>
     <li>Пункт1</li>
     <li>Пункт2</li>
     <li>Пункт3</li>
</ul>
e. А как выделить слово жирным шрифтом?
Выделить слово жирным шрифтом можно при помощи тегов <b> и <strong>. Нужно обернуть нужное слово или фразу в контейнер:
<b>важная фраза</b>
